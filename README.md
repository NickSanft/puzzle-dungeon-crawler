# Puzzle Dungeon Crawler

A roguelite dungeon crawler where puzzles replace enemies. Built in Godot 4 with GDScript, deployed to GitHub Pages.

## Concept

- Short runs (~15–20 min) through tile-based dungeons
- Rooms contain **nonogram** puzzles (scaling difficulty; color nonograms unlock later)
- Wrong cells on submit deal HP damage
- Solve puzzles to earn **Puzzle Glimbos** — persistent currency for unlock-based meta-progression
- Features: boss nonograms, daily seed, between-floor Glimbo shop

## Stack

- Godot 4.6 (GDScript)
- HTML5 / Web export
- Hosted on GitHub Pages (deployed via GitHub Actions)
